Experiment 2 - Vector Art - Shader Fun

Description

What are you trying to acomplish with this experiment?

I wanted to be able to write shader code for P5.js, and create intresting color variations with mouse control.

Technical

Describe in detail what you did technically and how it works.

By using WEBGL mode, I am able to write a Vertex and Fragment shader in GLSL languange and call the p5 function "shader()" which takes my shader as a parameter

Reflection

This project was due in part to me getting a large portion of the way through this assignment and having to start over. Not all was lost, because now I understand how shaders can be programmed into P5 sketches!

Experiment 2 - Interactivity - Most Valuable Letters

Description

What are you trying to acomplish with this experiment?

A typing visualization that emphasizes commonly used lexicons in a satisfying manner. Type words and watch the visualization. Note: Do not use the space bar, it scrolls the screen.

Technical

Describe in detail what you did technically and how it works.

By using an array and p5's unchar() function that turns the key input into its integer representation, I am able to keep track of how many times each letter has been typed, and increase its size accordingly, all while persisting even after other letters are typed.

Reflection

Each person that worked on the code should reflect on the process, the difficulties, and the successes of the experiment.

I really loved how this sketch turned out. It looks beautiful, and it uses a lot of coding habits that I carried over from working with C-like languages, which are my favorite to work with.
