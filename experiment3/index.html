Experiment 3 - Generative Methods - Type Your Own L-System!

Description

What are you trying to acomplish with this experiment?

I wanted to make an interactive grammar builder for an L-System that users can experiment with and share their results by sharing the keystrokes used to build their patterns.

Technical

Describe in detail what you did technically and how it works?

As the user types, their input is displayed at the top of the canvas, and is subsequently fed as input to the system of rules employed by my L-System. Each letter on the keyboard corresponds to either a combination of multiple rules or a singular rule which is then drawn to the canvas. Try pressing the same letter repeatedly for beautiful effects!
