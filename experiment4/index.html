Experiment 4 - Paint by Synthesizer

Description

What are you trying to acomplish with this experiment?

I wanted to create a fun, easy to use synthesizer that emphasized experimentation and play. Try playing Ode to Joy by typing: SSDFFDSAMMASSAA SSDFFDSAMMASAMM!

Technical

Describe in detail what you did technically and how it works?

I used the PolySynth module that is built into P5, and created a visualization that represents the current note being played and its ADSR.

Reflection

Each person that worked on the code should reflect on the process, the difficulties, and the successes of the experiment.

I had a lot of fun putting this experiment together, as working with synthesizers is a hobby of mine and I have always been curious as to how difficult it would be to program something like this up.
